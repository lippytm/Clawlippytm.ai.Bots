# System Architecture

## Role
`Clawlippytm.ai.Bots` is the public bot product and conversational systems hub in the wider lippytm ecosystem.

## Core Functions
- document bot product ideas and use cases
- support public bot demos and interaction patterns
- connect public-facing bot concepts to internal AI operations
- align product concepts with the public ecosystem map

## Ecosystem Position
```text
Public Ecosystem Hub
  -> Claw bot product hub
  -> connected to internal MyClaw ops and control tower standards
```

## Connected Repositories
- `lippytm.ai`
- `MyClaw.lippytm.AI-`
- `lippytm-lippytm.ai-tower-control-ai`

## Design Principles
- public clarity first
- modular bot concepts
- strong linkage to internal operations
- clear positioning between product, demo, and architecture docs

## Notes
Use this repo to present diversified bot concepts while keeping operational complexity separated into the internal MyClaw and control tower repos.
